# Goal: 생애 문제 컨설팅 루프 검증

상태: active
생성일: 2026-06-20

## 목표

사용자가 사주/기질 리포트를 읽고 끝나는 것이 아니라, 실제 고민을 AI 상담에 입력하고 코칭 액션으로 이어지는지 검증한다.

## 성공 기준

- 베타 사용자 5명 이상이 프로필 입력 후 리포트 1개 이상 생성.
- 그중 3명 이상이 AI 상담 질문 입력.
- 그중 2명 이상이 코칭 액션 등록 또는 직접 추가.
- 인터뷰에서 "다음 고민에도 다시 물어볼 의향"이 3명 이상에게서 확인.

## 주요 리스크

- 리포트가 길고 재밌지만 상담/행동으로 이어지지 않을 수 있다.
- 개인정보/LLM 데이터 사용 고지가 부족하면 외부 베타가 어렵다.
- Gemini Pro 비용이 실험 반복을 제한할 수 있다.
