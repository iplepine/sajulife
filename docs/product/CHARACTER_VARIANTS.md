## 사주언니 러프 후보

기존 `public/brand-icons/saju-unni.png`는 교체하지 않고 유지한다. 아래 이미지는 "헝클어진 머리, 안경, 막 입은 언니" 방향의 후보 자산이다.

| 파일 | 용도 메모 |
|---|---|
| `/brand-icons/saju-unni-rough/rough-unni-01-messy-bun-notebook.png` | 메인 캐릭터 후보. 부드러운 표정과 노트 소품. |
| `/brand-icons/saju-unni-rough/rough-unni-02-hoodie-notebook.png` | 작은 영역/아이콘 후보. 후드와 노트 중심. |
| `/brand-icons/saju-unni-rough/rough-unni-03-coffee-cardigan.png` | 피곤하지만 정확히 보는 언니 톤. 메인 후보로 가장 적합. |
| `/brand-icons/saju-unni-rough/rough-unni-04-seated-sweatshirt.png` | 가장 blunt한 상담자 톤. 빈 상태나 온보딩 후보. |
| `/brand-icons/saju-unni-rough/rough-unni-05-plaid-desk.png` | 생활감이 강한 작업실 톤. 큰 일러스트 영역 후보. |

## 적용 전 기준

- 홈 히어로에는 1장만 사용한다.
- 반복 카드나 메뉴 아이콘에는 캐릭터 이미지를 반복하지 않는다.
- 실제 교체 전에는 선택본 1장을 WebP/PNG로 경량화한 뒤 적용한다.
- 작은 아이콘으로 쓰려면 별도 크롭 버전을 만든다.
