# 디자인 가이드

마지막 갱신일: 2026-06-26

이 문서는 `사주언니 x 기질오빠` 컨셉을 제품 UI에 적용할 때의 기준이다. 목적은 캐릭터성은 살리되, 리포트 본문이 장난감처럼 보이지 않게 프리미엄 상담 문서의 밀도를 유지하는 것이다.

## 브랜드 톤

- 사주언니: 사주를 먼저 잡아주는 친근한 해석자. 말은 직설적이지만 사용자를 비웃지 않는다.
- 기질오빠: 기질과 행동 패턴을 정리해주는 현실적인 조력자. 차분하고 실용적인 느낌을 맡는다.
- 둘의 조합은 `사주언니 x 기질오빠`로 표기한다. UI에서는 곱하기 기호보다 `x`를 기본으로 쓴다.
- 배경은 따뜻한 아이보리 계열을 유지하고, 본문 텍스트는 검정/짙은 회색 중심으로 둔다.
- 캐릭터 이미지는 반복 노출하지 않는다. 리포트 상단 또는 대표 카드에서 한 번 강하게 쓰고, 반복 섹션에서는 텍스트와 컬러 신호로 구분한다.

## 리포트 섹션 컬러

리포트 하단 접힘 섹션은 모두 같은 붉은 포인트를 쓰면 반복 카드처럼 보인다. 섹션별 성격을 구분하기 위해 톤다운한 빨주노초파남보 계열의 포인트 컬러를 쓴다.

적용 원칙:

- 제목 전체 색을 바꾸지 않는다. 제목과 본문은 읽기 좋은 검정/회색을 유지한다.
- 컬러는 왼쪽 세로 막대, 작은 점, 얇은 상단 라인 같은 보조 신호에만 쓴다.
- 원색 무지개, 그라데이션, 과한 배경색은 쓰지 않는다.
- 같은 리포트 안에서 색은 섹션의 의미를 빠르게 스캔하게 하는 용도다. 감정 톤을 과장하지 않는다.
- 공유 페이지와 모바일에서도 같은 매핑을 유지한다.

권장 매핑:

| 섹션 계열 | 컬러 역할 | 권장 색 |
|---|---|---|
| 오행구성 / 기질구성 | 전체 에너지, 구성, 시작 | 초록 `#2F8B6B` |
| 기본성향 | 핵심 성향, 자기 이해 | 코랄 `#C95C55` |
| 직업운 | 실행, 일, 생산성 | 주황 `#D28B2D` |
| 금전운 / 가족금전운 | 돈, 자원, 관리 | 골드 `#C69A2A` |
| 인간관계운 / 가족분위기 | 관계, 연결, 대화 | 청록 `#2E9A8F` |
| 건강운 / 가족건강운 / 스트레스관리 | 회복, 몸, 조절 | 파랑 `#3F73B5` |
| 대운 / 가족대운 별 비교 / 인생 시기 | 큰 흐름, 장기 관점 | 남색 `#344F86` |
| 올해 실행전략 | 전략, 전환, 실행 계획 | 보라 `#7B5AA6` |

## 한줄문구 강조

- 개인 사주 상단 대표 문장은 AI 리포트 JSON의 `title`을 우선 사용한다. 아직 리포트가 없거나 옛 저장본처럼 `title`이 없으면 일간/계절/띠 계산값으로 만든 문구를 fallback으로 보여준다.
- 한줄문구는 말풍선이 아니라 리포트형 강조문으로 보인다.
- 큰따옴표를 실제 텍스트로 붙이지 않는다.
- 박스 테두리는 약하게 하거나 제거하고, 컬러 포인트는 왼쪽 막대 중심으로 둔다.
- 접힘 섹션 안에서는 카드 안 카드처럼 보이지 않게 배경 대비를 낮춘다.
- 섹션 요약은 모두 `~결`로 끝나지 않게 프롬프트에서 제한한다.

## 인생 시기 그래프 문구

- LifeCircle의 위/아래 축은 좋고 나쁨이 아니라 힘이 오는 방향을 설명하는 보조 기준이다. 리포트의 메인 해석처럼 크게 강조하지 않는다.
- `도움받는 시기 / 책임지는 시기`처럼 도덕 평가로 읽힐 수 있는 말은 피한다.
- 위쪽은 `주변에서 도와주는 시기`로 설명한다. 사람·환경·기회가 힘을 보태는 구간이라는 뜻이다.
- 아래쪽은 `내가 스스로 해내는 시기`로 설명한다. 선택·일·역할을 내가 잡고 가는 구간이라는 뜻이다.
- 이 축은 안내 박스 안의 읽는 법으로만 설명하고, 그래프 하단 요약 문장에서는 반복 강조하지 않는다.

## 구현 기준

- 주요 구현 위치: `components/ReportView.tsx`, `app/globals.css`
- 개인 사주 상단 대표 문장: `components/report/PersonalReportBody.tsx`
- 인생 시기 그래프 문구: `components/LifeCircle.tsx`
- 공유 렌더도 `ReportView`를 쓰므로 같은 색상 정책이 적용되어야 한다.
- 새 섹션이 추가되면 의미상 가장 가까운 섹션 계열 색을 재사용한다.
